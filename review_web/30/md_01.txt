COMO FUNCIONA UM SITE
 
	item antes do primeiro ponto - subdomínio - local específico do servidor
	depois do primeiro ponto - domínio - direciona para um servidor

	client - javascript, css, html	
	server - php - executado no servidor

COMO UM SERVIDOR FUNCIONA
	computador em algum lugar do mundo
	sites se hospedam em algum servidor(es)
	Precisa rodar um software que permita o acesso de fora à essa máquina

	Tipos de servidores web
		Apache 95% dos servidores
		ISS(Internet Information Services)
		Tomcat(java)
		JBoss(java)

	Como funciona
		O client pede algo ao servidor
		O servidor retorna o que foi pedido

COMO O PHP FUNCIONA
	Software instalado no servidor
	Interpretado pelo Apache 
	 -Quando o usuário acessa um site o apache é o primeiro que tem acesso a requisição - faz o processamento e chama o PHP

	client -> Apache -> PHP -> Server Filesystem/Database/Mail Server

PÁGINA ESTÁTICA VS PÁGINA DINÂMICA
	estática - conteúdo já criado - sem processamento - server->client
	Dinâmica - precisa de processamento - server<->db->client
	
	conteúdo estático - não depende de processamento, nem circunstâncias para rodar uma coisa ou outra
